# CodeQuest 🚀

Интерактивная веб-платформа для обучения программированию школьников 7–15 лет. Проект разработан в рамках хакатона (март 2026).

## 📖 О проекте

**CodeQuest** — это образовательная платформа, где обучение программированию превращается в увлекательное приключение. Пользователи проходят уроки, выполняют мини-проекты, проходят тесты и отслеживают свой прогресс. Интерфейс выполнен в ярком, мотивирующем стиле.

### Ключевые особенности

- 🎓 **Обучение через практику**: каждый урок сочетает теорию, практические задания и тесты
- 👥 **Ролевая система**: пользователи, модераторы, администраторы
- 📊 **Отслеживание прогресса**: визуализация достижений и шкала прогресса
- 🛡️ **Родительский контроль**: мониторинг активности детей
- 💬 **Сообщество**: чат и рейтинги для соревновательного элемента
- 🎮 **Геймификация**: награды и достижения за успехи

## 🛠 Технологии

- **Python 3.8+**
- **Flask 3.0.0** — микрофреймворк
- **Flask-SQLAlchemy 3.1.1** — ORM для работы с БД
- **Flask-Login 0.6.3** — управление аутентификацией
- **Flask-WTF 1.2.1** — работа с формами и валидация
- **Werkzeug 3.0.1** — утилиты WSGI
- **Bootstrap 5** — адаптивный дизайн
- **SQLite** — база данных (легко переключается на PostgreSQL)

## 📋 Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- git

## 🎮 Модули платформы

### Модуль 1: Аутентификация и пользователи
- Регистрация с проверкой email
- Роли: студент, учитель, родитель, администратор
- Панель администратора для управления пользователями

### Модуль 2: Конструктор уроков
Каждый урок содержит:
- 📖 **Теория** — текст с примерами кода
- 💻 **Практика** — интерактивные задания и мини-проекты
- ✅ **Тест** — викторина для проверки знаний

### Модуль 3: Отслеживание прогресса
- Визуальная шкала прогресса по урокам
- Система достижений и наград
- Магазин для обмена наград

### Модуль 4: Сообщество
- Общий чат для общения
- Таблица лидеров (рейтинги)
- Возможность делиться проектами

### Модуль 5: Родительский контроль
- Отслеживание активности ребенка
- Статистика по пройденным урокам и тестам
- Уведомления о достижениях

## 🔧 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| `ModuleNotFoundError: No module named 'flask'` | Активируйте виртуальное окружение: `source venv/bin/activate` (Linux/macOS) или `venv\Scripts\activate` (Windows) |
| `RuntimeError: A secret key is required` | Добавьте `SECRET_KEY` в файл `.env` |
| Ошибка порта 5000 | Запустите на другом порту: `flask run --port=5001` |
| База данных не создается | Проверьте права на запись в директорию проекта |
| Ошибка импорта `app` | Убедитесь, что находитесь в корневой директории проекта |

## 🏭 Запуск в продакшене

### С использованием Gunicorn

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:8000 app:app
```

### С PostgreSQL

1. Установите драйвер:
```bash
pip install psycopg2-binary
```

2. Измените `config.py` или переменную окружения:
```env
DATABASE_URL=postgresql://username:password@localhost/codequest
```
